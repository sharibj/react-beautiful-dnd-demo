# Drag Drop Demo
## setup
```
npm init -y
npm install react react-dom react-beautiful-dnd
npm install --save-dev @babel/core @babel/preset-env @babel/preset-react babel-loader webpack webpack-cli
```
## build
```
npx webpack --mode=development
```
## initial files
- webpack.config.js
- .babelrc
- /src/index.js
- index.html
